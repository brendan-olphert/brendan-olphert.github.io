Margin, Border, and Padding

Margin

Margin is the CSS property that defines the space outside of an element to its next outside element. Margin affects elements that both have or do not have borders. If an element has a border, margin defines the space from this border to the next outer element. If an element does not have a border, then margin defines the space from the element content to the next outer element.

Border

Border is the CSS property that allows you to specify the style, width and colour of an elements border. It will create a defined border with the elements padding on the inside and the elemnts margins on the outside. Border style, width, and colour can be customised. Like margins individual values can be set for the top, right, bottom and left sides of the border. a complete list of border values can be found here

Padding

Padding is the CSS property that defines the space between an element content and its border. If an element has a border around it, padding will give space from that border to the element content which appears in that border. If the element does not have a border padding will have no effect. There are properties for setting the padding for each side of an element (top, right, bottom, and left).

Difference Between Padding,Margin and Border

A key difference between margin and padding is that while padding deals with the inner space, margin deals with the outer space to the next outer element. margin helps to create space around the element outside the border, while padding helps to create space around the element inside the border. The margin is intended for providing space between outside HTML elements or the sides of the page. Padding is used for providing visual space between the content and the border of the box. using margin will not affect your element's dimensions whereas padding will add to your elements dimensions (even without a border)
