Describe what the DOM is and an example of how you might interact with it.

The Document Object Model (DOM) is an underlying hierachal structure that allows used to allow interactivity and interface between Java-Script, HTML, and CSS. The DOM represents the document as Nodes, each of which relate to and element (or attribute etc) in the HTML or CSS

An example of how you might interact with the DOM is through use of the developer tools functionality in browsers such as Chrome

Explain the difference between accessing data from arrays and objects.

The properties in an object can be accessed with dot and bracket notation e.g. object.key[value]

The properties in an array are arranged using zero-based indexing and can be accessed by calling their number in the list hierachy

Explain what functions are and why they are useful.

Functions are alot like brackets in mathematics, they group or wrap parts of code. Arguments and code contained within the wrap or group are excecuted as a group by the Java-Script engine. I have been thinking it like if your code is a list of instructions for your computer to read, a function is like a self contained task for the engine to carry out before moving on down the tree.

Functions are incredibly useful. They carry out tasks that save alot of code.
